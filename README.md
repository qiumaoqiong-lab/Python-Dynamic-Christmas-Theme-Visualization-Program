# 🎄 Python 动态圣诞主题可视化程序

一个用 Python Turtle 实现的动态圣诞树可视化项目，绘制夜晚星空下的圣诞树效果。

## ✨ 项目效果
![项目效果截图](images/tree.png.png)

## 🛠️ 运行环境
Python 3.x
无需额外安装依赖（仅使用标准库 turtle 和 random）

## 🚀 运行方法
1. 下载或克隆本项目
2. 运行主程序文件：
   ```bash
   python christmas_tree.py
   ```

## 📝 说明
程序会自动绘制分形结构的圣诞树、彩色装饰、星空和月亮
每次运行装饰位置和星星会有随机变化
